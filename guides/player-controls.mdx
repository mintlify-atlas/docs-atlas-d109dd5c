The Ripper **reproductor** (player) page provides comprehensive controls for playing music tracks with synchronized video sheet music. Access the player by clicking any track card from the music browser.

## Player Interface

### Track Information Header

At the top of the player, you'll see:

```html
<div class="label-tag">En reproducción</div>
<h1 class="title-main" id="txt-titulo">Cargando...</h1>
<div class="meta-row">
    <span class="meta-item" id="txt-banda">--</span>
    <span class="meta-dot"></span>
    <span class="meta-item">Tonalidad: <span class="tono-badge" id="txt-tono">--</span></span>
</div>
```

- **Status badge**: "En reproducción" (Now Playing) with animated indicator
- **Track title**: Large, gradient-styled heading
- **Artist name**: Displayed alongside the tonality
- **Tonality badge**: Color-coded based on the musical key

### Video Container

The player supports two video sources:

<Tabs>
  <Tab title="YouTube">

    ```javascript
    window.player = new YT.Player("player", {
        height: "100%",

            rel: 0,

        }
    ```

    YouTube videos have custom controls disabled (`controls: 0`) and use Ripper's custom control panel.
  </Tab>
  
  <Tab title="Google Drive">
    Embedded Google Drive videos using iframe preview:
    
    ```javascript
    const iframe = document.createElement("iframe");
    iframe.src = `https://drive.google.com/file/d/${temaData.videoId}/preview`;
    iframe.allow = "autoplay";
    ```
    
    <Warning>
    Google Drive videos have limited control functionality. The external open button is disabled via a corner overlay layer.
    </Warning>
  </Tab>
</Tabs>

## Control Panel

### Play/Pause Button

The central red circular button controls playback:

```html
<button id="btn-play" onclick="togglePlay()" aria-label="Reproducir o pausar">
    <svg class="icon-play" viewBox="0 0 24 24"><path d="M8 5v14l11-7z"/></svg>
    <svg class="icon-pause" style="display:none" viewBox="0 0 24 24">
        <path d="M6 19h4V5H6v14zm8-14v14h4V5h-4z"/>
    </svg>
</button>
```

**Features:**
- Animated pulse ring when paused
- Icon switches between play ▶ and pause ⏸
- Hover effect with scale and shadow
- Click to toggle playback state

```javascript
window.togglePlay = () => {
    if (!isYouTube || !window.player) return;
    playing ? window.player.pauseVideo() : window.player.playVideo();
};
```

<Note>
The play/pause button only functions for YouTube videos. Google Drive videos use native player controls.
</Note>

### Seek Buttons

Precise time navigation with multiple intervals:

```html
<button class="btn-seek" onclick="seek(-10)">−10s</button>
<button class="btn-seek" onclick="seek(-5)">−5s</button>
<button class="btn-seek" onclick="seek(-1)">−1s</button>
<!-- Play button -->
<button class="btn-seek" onclick="seek(1)">+1s</button>
<button class="btn-seek" onclick="seek(5)">+5s</button>
<button class="btn-seek" onclick="seek(10)">+10s</button>
```

Seek implementation:

```javascript
window.seek = (s) => {
    if (isYouTube && window.player) {
        window.player.seekTo(window.player.getCurrentTime() + s, true);
    }
};
```

### Restart Button

Return to the track's designated start time:

```javascript
window.reiniciar = () => {
    if (isYouTube && window.player) {
        window.player.seekTo(inicio, true);
        window.player.playVideo();
    }
};
```

The `inicio` value comes from:
1. URL parameter `t` if provided: `?t=45`
2. Track's `inicio` field in database
3. Default: 0 (start of video)

```javascript
const forcedStart = Number(params.get("t") || "0");
inicio = (!Number.isNaN(forcedStart) && forcedStart > 0) 
    ? forcedStart 
    : (temaData.inicio || 0);
```

<Tip>
You can share timestamped links by copying the link while the video is playing - the current time will be preserved in the `t` parameter.
</Tip>

## Progress Tracking

### Visual Progress Bar

```html
<div class="progress-track">
    <div class="progress-fill" id="progress-fill"></div>
</div>
```

Updated every 500ms:

```javascript
setInterval(() => {
    if (window.player && isYouTube) {
        const d = window.player.getDuration?.() || 0;
        const c = window.player.getCurrentTime?.() || 0;
        txtCurrent.textContent = fmtTime(c);
        txtTotal.textContent = fmtTime(d);
        if (d > 0) progressFill.style.width = (c / d * 100) + "%";
    }
}, 500);
```

### Time Display

Formatted as MM:SS:

```javascript
function fmtTime(seconds) {
    const total = Math.max(0, Math.floor(seconds || 0));
    const m = Math.floor(total / 60);
    const s = total % 60;
    return `${String(m).padStart(2, "0")}:${String(s).padStart(2, "0")}`;
}
```

Displayed as:
```html
<div class="time-row">
    <span id="txt-time-current">00:00</span>
    <span id="txt-time-total">00:00</span>
</div>
```

## Playback Speed Control

### Speed Selection

<Warning>
Playback speed control is **only available for YouTube videos**. The speed selector is hidden for Google Drive videos.
</Warning>

```html
<select id="speed-select" class="speed-select" aria-label="Velocidad de reproducción">
    <option value="0.75">0.75x</option>
    <option value="1" selected>1x</option>
    <option value="1.25">1.25x</option>
    <option value="1.5">1.5x</option>
    <option value="2">2x</option>
</select>
```

Speed change handler:

```javascript
speedSelect.addEventListener("change", () => {
    if (!isYouTube || !window.player) return;
    const rate = Number(speedSelect.value);
    if (!Number.isNaN(rate)) window.player.setPlaybackRate(rate);
});
```

## Additional Controls

### Copy Link Button

Share the current track with timestamp:

```javascript
btnCopyLink.addEventListener("click", async () => {
    try {
        let url = window.location.href;
        if (isYouTube && window.player) {
            const t = Math.floor(window.player.getCurrentTime?.() || 0);
            const u = new URL(window.location.href);
            u.searchParams.set("t", String(t));
            url = u.toString();
        }
        await navigator.clipboard.writeText(url);
        btnCopyLink.textContent = "Copiado";
        setTimeout(() => { btnCopyLink.textContent = "Copiar link"; }, 1200);
    } catch {
        btnCopyLink.textContent = "No disponible";
    }
});
```

The copied URL includes:
- Track ID: `?id=abc123`
- Category: `&cat=huayno`
- Current timestamp: `&t=45` (if playing YouTube video)

### Fullscreen Toggle

Expand video to fullscreen:

```javascript
btnFullscreen.addEventListener("click", async () => {
    const container = document.getElementById("video-container");
    if (!document.fullscreenElement) {
        try {
            await container.requestFullscreen();
        } catch (e) {
            console.error(e);
        }
    } else {
        await document.exitFullscreen();
    }
});
```

<Tip>
In fullscreen mode, all keyboard shortcuts continue to work. Press ESC or click the fullscreen button again to exit.
</Tip>

Shown when video is paused:

```html
<div id="yt-overlay">
    <span class="ov-icon">🎵</span>
    <span class="ov-title">Presiona Play</span>
    <span class="ov-sub">Usa el botón rojo de abajo para reproducir</span>
```

Automatically hidden when playing:

```javascript
if (e.data === YT.PlayerState.PLAYING) {
    btn.classList.add("playing");
    playing = true;
    document.getElementById("yt-overlay").style.display = "none";
}
```

### Tab Visibility Handling

Video automatically resumes when returning to tab:

```javascript
document.addEventListener("visibilitychange", () => {
    if (document.visibilityState === "visible" && isYouTube && window.player && playing) {
        ignorarPausa = true;
        setTimeout(() => { ignorarPausa = false; }, 2000);
        window.player.playVideo();
    }
});
```

<Note>
This prevents the overlay from appearing when you switch back to the Ripper tab after the browser auto-pauses the video.
</Note>

## YouTube vs Google Drive Differences

| Feature | YouTube | Google Drive |
|---------|---------|-------------|
| Custom Controls | ✅ Full control panel | ❌ Limited (native only) |
| Seek Buttons | ✅ All intervals | ❌ Not available |
| Speed Control | ✅ 0.75x - 2x | ❌ Not available |
| Progress Bar | ✅ Live updates | ❌ Not available |
| Keyboard Shortcuts | ✅ All shortcuts | ❌ Not available |
| Restart Function | ✅ Custom start point | ❌ Not available |
| Timestamp Links | ✅ Preserved in URL | ⚠️ Link only (no time) |

```javascript
if (temaData.fuente === "youtube") {
    isYouTube = true;
    document.getElementById("panel-controles").style.display = "flex";
    // ... YouTube setup
} else {
    speedWrap.style.display = "none";
    document.getElementById("capa-drive").style.display = "block";
    document.getElementById("aviso-drive").style.display = "block";
    // ... Drive setup
}
```
