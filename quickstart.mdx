Get your account set up and start streaming Peruvian huayno music with synchronized sheet music in just a few steps.

## Prerequisites

Before you begin, you'll need:

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An active internet connection
- A WhatsApp account to request access

<Note>
  Ripper requires **account approval** from the administrator. Each account is limited to **one active device at a time** for security purposes.
</Note>

    Enter your email and password in the login panel.
    
    From `index.html:416-420`:

    2. A session is claimed for your device (expelling any other active sessions)
    3. You're redirected to the home page

    - **Tonal sorting**: Songs automatically sorted by tonality, then by major/minor

## Common Login Errors

<AccordionGroup>
  <Accordion title="'Acceso denegado. Verifica tus datos.'">
    **Cause**: Invalid email or password, or account doesn't exist.
    
    **Solution**:
    - Double-check your email and password for typos
    - Ensure your account has been created by the administrator
    - Contact support via WhatsApp if issues persist
  </Accordion>
  
  <Accordion title="Session displacement / 'Sesión desplazada'">
    **Cause**: You logged in from another device or browser.
    
    **Solution**:
    - Ripper enforces **single-device sessions**
    - If you see the expulsion screen, another device logged in with your credentials
    - Log out from other devices before logging in again
    
    From `js/seguridad.js:41-71`:
    ```javascript
    export function manejarExpulsion(contenedor, auth) {
        contenedor.innerHTML = `
            <div class="expulsion-card">
                <h3>Sesión desplazada</h3>
                <p>Tu cuenta fue abierta en otro dispositivo o navegador.</p>
                <div class="aviso-importante">
                    ⚠️ <strong>Recuerda:</strong> Solo se permite <b>un dispositivo activo</b> 
                    por cuenta a la vez.
                </div>
                <p>Cerrando sesión en <span id="seg-cuenta">10</span>s...</p>
            </div>`;
        // 10-second countdown then auto-logout
    }
    ```
  </Accordion>
  
  <Accordion title="Blank page after login">
    **Cause**: JavaScript error or network connectivity issue.
    
    **Solution**:
    - Check browser console for errors (F12 → Console tab)
    - Ensure you have a stable internet connection
    - Clear browser cache and try again
    - Try a different browser
  </Accordion>
</AccordionGroup>

## Admin Access

If your account has admin privileges (`admin@ripper.com`), you'll see an **Admin** button in the header on `home.html`:

From `home.html:379-390`:
```javascript
onAuthStateChanged(auth, (user) => {
    if (user && user.email === ADMIN_EMAIL) {
        const zone = document.getElementById("admin-zone");
        if (zone) {
            zone.innerHTML = `
                <a href="admin.html" class="btn-nav btn-admin">
                    <svg>...</svg>
                    <span>Admin</span>
                </a>`;
        }
    }
});
```

<Warning>
  Admin access is restricted to the designated administrator email. Regular users will not see the Admin button.
</Warning>
